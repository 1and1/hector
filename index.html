hector by rantav

a high level client for cassandra

Hector is a java client for the distributed nosql Cassandra database. Hector is the de-facto client used by many java developers to speak to cassandra. A partial feature list includes: - high level and object oriented interface shadowing the rough thrift API - automatic failover support - JMX health and performance counters All recent cassandra versions are supported: 0.5.0, 0.5.1 and 0.6.0

Dependencies

Cassandra and thrift. Full list is in the pom.xml or build.xml files.
